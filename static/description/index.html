Elastic Integration (SFTP)

Two-way Odoo / Elastic B2B integration via SFTP flat files.

What it does

Outbound exports

  • Products & variants (with color/size attributes)
  • Customers, custom fields, sales reps and rep mappings
  • Catalogs and catalog mappings
  • Inventory by warehouse
  • Pricing per pricelist (multi-tier)

Inbound imports

  • Order files staged for review before becoming sale orders
  • Auto-archived after processing
  • Per-row error tracking with one-click retry
  • Customer cross-reference table for Sold-To / Ship-To matching

Configurable pricing

Flag any pricelist with Send to Elastic to publish it as a price group. Set an Elastic Price Group Code (LP, D, PL...) or let the module derive one from the pricelist name. With no pricelist flagged, the export falls back to the product list price.

Beta & Production environments

Maintain separate SFTP connections for sandbox and live environments and switch with a single click. Each connection has its own host, credentials, and directory layout.

Security & auditing

  • SSH key or password authentication
  • Dedicated Elastic User and Elastic Manager permission groups
  • Export and import logs with row counts, filenames, and error detail
